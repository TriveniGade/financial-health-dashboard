# financial-health-dashboard
Project Overview:

This project is an interactive Financial Health Dashboard developed using Power BI to analyze an organization’s financial performance.

Features:
-Income statement analysis
-Balance sheet monitoring
-Cash flow tracking
-Profitability trends
-Forecasting and budgeting

Tools Used:
-Power BI
-Excel
-DAX
-Power Query

Dashboard Pages:
1.Financial Overview
2.Balance Sheet Analysis
3.Cash Flow Dashboard
4.Forecast Analytics

Key Insights:
-Revenue growth trends
-Expense analysis
-Net profit tracking
-Cash flow forecasting

Files Included:
-Power BI Dashboard (.pbix)
-Excel Dataset
-Dashboard Screenshots
